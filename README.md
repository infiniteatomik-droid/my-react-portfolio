# 🚀 Моё Портфолио Разработчика на React + Tailwind CSS

Добро пожаловать в репозиторий моего практического мини-проекта, созданного на экваторе интенсивного обучения фронтенд-разработке!

🔗 **Живая ссылка на сайт:** [Посмотреть моё портфолио](https://infiniteatomik-droid.github.io/my-react-portfolio/)

## 🛠 Технологический стек проекта
* **Библиотека:** React.js (управляемые формы, хуки `useState`, `useNavigate`)
* **Роутинг:** `react-router-dom` (многостраничная навигация без перезагрузки)
* **Стилизация:** Tailwind CSS (адаптивная сетка Grid, Flexbox, современные градиенты)
* **Деплой:** GitHub Pages
